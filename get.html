Getting Codedit

Hosted

A prebuilt version of Codedit is hosted by following CDNs:

Unpkg

<script src="https://unpkg.com/codedit@1.0.6/codedit.min.js"></script>

jsDelivr

<script src="https://cdn.jsdelivr.net/npm/codedit@1.0.6/codedit.min.js"></script>

Properties

You can always go with the default Codedit:

<cd editable="false" ><!-- code --></cd>

Specify a language! (leave blank for auto-detect)

<cd lang="js" >function doSomething() {}</cd>

Style however you want! class="elevated"

.elevated { box-shadow: 0 1px 6px 0 rgba(32, 33, 36, .28) }
